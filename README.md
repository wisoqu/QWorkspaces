# QWorkspaces

QWorkspaces - это не еще один todo-лист. Это заход на персональное цифровое рабочее место, где у пользователя есть собственное пространство для планирования, задач, заметок, напоминаний и постоянный личный ИИ-ассистент рядом.

Сейчас проект находится на пути к MVP. Реализовано совсем немного: базовая логика фронтенда приветственного экрана и стартовая структура приложения на Flet. Но сам замысел у проекта гораздо шире: сначала собрать сильную локальную версию, а затем вырастить из нее полноценный продукт с амбициозной бизнес-моделью.

## В чем идея

QWorkspaces задумывается как единая рабочая среда для человека, а не как набор разрозненных экранов.

Пользователь должен получить:
- личное пространство для организации своей работы и жизни;
- задачи, заметки и напоминания в одном месте;
- календарное планирование с синхронизацией с Google Calendar;
- ИИ-ассистента, который не просто отвечает, а помогает выстраивать рутину и принимать решения;
- память о контексте пользователя, привычках, договоренностях и приоритетах.

Главная ставка проекта - не на список дел, а на ощущение, что у пользователя появилось свое умное рабочее место, которое понимает контекст, помнит важное и помогает держать фокус.

## Что есть сейчас

Текущее состояние проекта честное и раннее:
- есть базовая структура Flet-приложения;
- есть приветственный экран;
- есть простая маршрутизация между экранами;
- есть заготовка под личное пространство пользователя;
- проект движется к первой MVP-версии.

На данный момент это именно фундамент, а не готовый продукт.

## Что планируется в MVP

Ближайшая цель - собрать локальную рабочую версию, в которой будет основа пользовательского пространства:
- личный экран пользователя;
- заметки и задачи;
- базовые напоминания;
- первичная структура рабочего пространства;
- подготовка архитектуры под будущего ИИ-ассистента.

Локальный этап нужен не как временная игрушка, а как способ собрать ядро продукта, отточить логику сценариев и понять, как пользователь реально взаимодействует со своим рабочим пространством.

## Куда проект идет дальше

Дальше QWorkspaces должен перерасти в гораздо более сильную систему:
- календарь с синхронизацией с Google;
- личные заметки, задачи и напоминания, связанные между собой;
- ИИ-модель, которая помогает планировать день и неделю;
- диалог с ассистентом о том, что, когда и почему лучше поставить в план;
- личная память ассистента, которая хранит детали о пользователе;
- периодическая сверка актуальности этой памяти через беседу;
- развитие в сторону супераппа с сильной продуктовой и бизнесовой моделью.

Идея в том, чтобы пользователь не просто вел список дел, а работал внутри системы, которая понимает его контекст и становится реальным персональным помощником.

## Почему это не просто todo

Обычный todo хранит задачи.

QWorkspaces должен:
- помнить контекст;
- помогать принимать решения;
- напоминать вовремя;
- связывать задачи, заметки и события;
- вести пользователя через диалог, а не через сухой список пунктов.

Это попытка собрать не менеджер задач, а персональный operating system layer для человека.

## Запуск

1. Создать и активировать виртуальное окружение.
2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Запустить приложение:

```bash
python src/main.py
```
